# تثبيت Alusus على ويندوز وتشغيل Hello World

يشرح هذا الدليل طريقة عملية لتثبيت الوسس على Windows ثم تجربة برنامج Hello World بسيط.

## الخطوة 1: تنزيل Alusus لويندوز

افتح صفحة التنزيل الرسمية واختر حزمة Windows. وتشير التعليمات الرسمية الحالية إلى:
1. تنزيل الحزمة،
2. فك ضغطها داخل مجلد،
3. ثم تشغيل ملف التثبيت المرفق.

## الخطوة 2: تشغيل المثبّت

بعد فك الضغط، شغّل الملف:

```text
setup.exe
```

ثم أكمل خطوات التثبيت.

## الخطوة 3: التحقق من التثبيت

افتح Command Prompt أو PowerShell ثم نفّذ:

```powershell
alusus --help
```

إذا لم يُعرف الأمر، أغلق الطرفية وافتحها من جديد وتأكد من نجاح التثبيت.

## الخطوة 4: إنشاء ملف Hello World

أنشئ ملفًا باسم `hello.alusus` بالمحتوى التالي:

```alusus
import "Srl/Console.alusus";
use Srl;

Console.print("Hello World!\n");
```

## الخطوة 5: تشغيل الملف

من داخل المجلد الذي يحتوي الملف:

```powershell
alusus hello.alusus
```

النتيجة المتوقعة:

```text
Hello World!
```

## تنظيم مقترح للمجلد

```text
C:\AlususProjects\HelloWorld\
  hello.alusus
```

## حل المشاكل الشائعة

### الأمر `alusus` غير معروف
أغلق الطرفية وافتحها من جديد بعد التثبيت. وإذا لزم الأمر، راجع مسار التثبيت الذي أنشأه البرنامج.

### امتداد الملف خاطئ
تأكد من أن الملف محفوظ بامتداد `.alusus` وليس `.txt`.

### لا يظهر خرج
تحقق من محتوى الملف ومن أنك تشغّل الأمر من داخل المجلد الصحيح.

## الخطوات التالية

بعد Hello World:
- اقرأ بوابة التوثيق الرسمية،
- تعلّم المتغيرات والحلقات والسلاسل النصية،
- ابنِ برنامجًا صغيرًا لسطر الأوامر،
- ثم جرّب تطوير الويب في Alusus.
