# مستعرض الوثائق - نسخة عربية

هذه نسخة من التطبيق نفسه لكن بصياغة عربية في Alusus وواجهة عربية بالكامل.

## التشغيل

```bash
alusus app.alusus
```

ثم افتح:

```text
http://localhost:8000/
```

## تسجيل الدخول

- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

## الملاحظات

- المنطق لم يتغير.
- ما زال التخزين داخل الذاكرة.
- الواجهة أصبحت عربية و `dir="rtl"`.
- صياغة ملف Alusus تعتمد على الكلمات المفتاحية العربية مثل:
  - `اشمل`
  - `استخدم`
  - `صنف`
  - `عرف`
  - `دالة`
  - `إذا`
  - `لكل`
  - `أرجع`
  - `وحدة`
  - `@beEndpoint`
  - `@مسار_موارد`
